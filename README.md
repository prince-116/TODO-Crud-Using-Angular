# TODO/Crud Using Angular

Simple crude using mogo db

# Step-1:

<p>open two terminals using:</p> 
<p>     1) cd client</p> 
<p>     2) cd server</p>

# Step-2:

<p>In Client Side terminal run: </p>
<p>    1) npm install </p>
<p>    2) npm start / ng serve</p>

<p>// running on port <a>http://localhost:4200/</a> </p>
